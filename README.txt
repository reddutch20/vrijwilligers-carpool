Vrijwilligers Carpool App - Installatiehandleiding

1️⃣ Installeer Node.js (https://nodejs.org, kies LTS versie)
2️⃣ Open een map en pak hier dit project uit
3️⃣ Open de map in een terminal en voer uit:
   npm install
4️⃣ Start de app met:
   npm start

De app opent dan in je browser op http://localhost:3000
